## 2. DESCRIPCIÓN DEL SERVICIO

TikBoosTTS es una aplicación móvil para Android que proporciona las siguientes funcionalidades:

### 2.1 Funciones Principales

- **Conexión a TikTok Live**: Permite conectarse a transmisiones en vivo de TikTok para recibir eventos en tiempo real
- **Texto a Voz (TTS)**: Convierte comentarios de usuarios en audio mediante tecnología de síntesis de voz
- **Alertas de Audio**: Sistema configurable de alertas sonoras para eventos específicos (seguidores, regalos, suscripciones)
- **Sistema de Cuotas**: Gestión de mensajes TTS mediante cuotas diarias y recompensas por anuncios
- **Configuración Personalizada**: Ajustes de voz, velocidad, tono, volumen y formato de comentarios
- **Sistema Anti-Spam**: Filtrado de mensajes y bloqueo de usuarios

### 2.2 Limitaciones del Servicio

- La Aplicación depende de servicios de terceros (TikTok, Google, Firebase)
- No garantizamos disponibilidad ininterrumpida del servicio
- Las funcionalidades pueden cambiar sin previo aviso
- El rendimiento puede variar según la conexión a internet y dispositivo

## 5. SISTEMA DE MENSAJES TTS Y CUOTAS

### 5.1 Cuota Diaria Gratuita

- **100 mensajes gratuitos por día** para conversión de texto a voz
- Las cuotas se resetean diariamente a las 00:00 hora local
- Los mensajes no utilizados NO se acumulan para el día siguiente

### 5.2 Mensajes Adicionales Mediante Anuncios

- Puede obtener **+200 mensajes** por cada anuncio recompensado completo
- Debe ver el anuncio completo para recibir la recompensa
- Los anuncios son proporcionados por Google AdMob
- No hay límite en la cantidad de anuncios que puede ver por día
- Los mensajes obtenidos por anuncios NO expiran hasta ser consumidos

### 5.3 Consumo de Mensajes

- Cada comentario convertido a voz consume **1 mensaje** de su cuota
- Las pruebas de audio en la configuración NO consumen mensajes
- Si no hay mensajes disponibles, el TTS no funcionará hasta obtener más
- El contador de mensajes se actualiza en tiempo real

### 5.4 Limitaciones

- No ofrecemos reembolsos por mensajes no utilizados
- El sistema de cuotas puede ser modificado con previo aviso
- Nos reservamos el derecho de ajustar las cuotas en caso de abuso
